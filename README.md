# Live Environmental Intelligence Engine

Real-time environmental anomaly detection prototype using streaming simulation and machine learning.

## Overview

This project explores real-time environmental anomaly detection prototype using simulated sensors data and streaming-based processing.

## Our Objective

To simulate environmental data streams and detect abnormal pollution patterns using machine learning.</br>
This prototype will be extended to integrate with real-world environmental IoT sensors in future stages.

## Planned Architecture

- Environmental data simulator
- Streaming loop
- Anomaly detection module
- Console-based alert output

## Current Status

This is the Initial project setup. We are currently focusing on Core modules to be implemented.</br>
We will update it constantly.

## Notes

- The `main.py` script uses the `pathway` library for streaming. The real `pathway` package does not support Windows; the code will run correctly in a compatible Linux/macOS environment with the library installed.
